# Parkir Calc

Kalkulator biaya parkir.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4

## Cara Menjalankan

```bash
# Install dependencies
npm install

# Menjalankan development server
npm run dev

# Build production
npm run build

# Preview hasil build
npm run preview

# Linting
npm run lint
```
